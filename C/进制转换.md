## C/C++

将十进制数转化为其他进制的数

```c
// num 十进制数
// str char数组, 用来存储结果
// n 代表准备转换的进制
itoa(num, str, n);
```

```c++
#include<stdio.h>
#include<stdlib.h>

// 注意必须调用stdlib.h函数库
int main(void) {
    int a = 15;
    char str[30];

    // 将15转为2进制
    itoa(a, str, 2);
    printf("%s\n", str);

    // 将15转为16进制
    itoa(a, str, 16);
    printf("%s\n", str);
    return 0;
}
```

## Python

自带函数 bin(), oct(), hex()

```py
x = 15

print("{0}的二进制值为:{1}".format(x,  bin(x)))

print("{0}的八进制值为:{1}".format(x,  oct(x)))

print("{0}的十六进制值为:{1}".format(x,  hex(x)))
```
