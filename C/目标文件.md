## C/C++目标文件

目标文件指源文件经过编译器预处理, 编译, 汇编过程后得到的.o 文件




### 生成

源文件main.c

```c
// 默认生成main.o
gcc -c main.c

// 指定.o文件名
gcc -c main.c -o main.o
```
