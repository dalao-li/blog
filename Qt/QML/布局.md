# 布局

## SplitView

SplitView将视图切割成不同的小块，每个小块可以是不同的组件

## 规则

### 水平布局

设置组件宽度

```js
implicitWidth: 200
```

### 垂直排列

设置组件高度

```js
implicitHeight: 200
```
