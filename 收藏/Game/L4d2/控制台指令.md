# L4d2控制台指令

## 基础指令


### 开启作弊模式

```sh
Sv_Cheats 1
```

### 无敌

```sh
god 1
```

### 穿墙

```sh
noclip
```

### 无限弹药不换弹夹

```sh
sv_infinite_ammo 1
```


### 设置第三人称

```sh
thirdpersonshoulder 1
```

- 绑定到按键 <kbd>g</kbd>

```sh
bind g "thirdpersonshoulder 1"
```
