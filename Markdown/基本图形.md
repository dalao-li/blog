## 连接

| 指令                        | 连接方式         |
| --------------------------- | ---------------- |
| A --> B<br>A -- 描述 --> B  | A箭头指向B       |
| A -- B<br>A -- 描述 -- B    | A不带箭头指向B   |
| A -.- B<br>A -. 描述 .- B   | A虚线指向B       |
| A -.-> B<br>A -. 描述 .-> B | A带箭头虚线指向B |
| A ==> B<br>A == 描述 ==> B  | A加粗箭头指向B   |
```mermaid
graph TB;
    A[A]-->B[B]
    A1[A]---B1[B]
    A4[A] -.- B4[B] 
    A5[A] -.-> B5[B] 
    A7[A] ==> B7[B] 
```

```mermaid
graph TB
    A[A] -- 描述 --- B[B] 
    A2[A] -- 描述 --> B2[B] 
    A5[A] -.描述.- B5[B] 
    A3[A] -. 描述 .-> B3[B] 
    A4[A] == 描述 ==> B4[B] 
```
