## 变量

- 定义
  
```sh
a=123
```

- 只读变量
  
```sh
readonly 变量
```

- 使用
  
```sh
${变量}
```

- {}与()的区别

`()`将内部的命令置于子Shell中执行


`{}`则是在同一个Shell内完成

 
例，获取VLC的pid，关闭VLC

```sh
#!/bin/bash
pid=${ps -aux | grep vlc | grep -v grep | awk '{print $2}'}

if ! "${pid}";then
    sudo kill -9 ${pid}
fi
```
