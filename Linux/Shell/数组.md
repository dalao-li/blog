## 初始化

数组用括号来表示，元素用空格符号分割开

- 括号定义
  
```sh
array=(v1 v2 ......)
```

- 下标定义
  
```sh
array[0]=v1
array[1]=v2
......
```

## 读取

- 单个读取
  
```sh
${array[index]}
```

- 全部获取

```sh
${array[*]} 或 ${array[@]}
```

- 获取长度
```sh
${#array[*]} 或 ${#array[@]}
```
