## 初始化

数组用括号来表示，元素用空格符号分割开

```sh
# 括号定义
array=(v1 v2 ......)

# 下标定义
array[0]=v1
array[1]=v2
......
```

## 读取

```sh
# 单个读取
${array[index]}

# 全部获取
${array[*]} 或 ${array[@]}

# 获取长度
${#array[*]} 或 ${#array[@]}
```
