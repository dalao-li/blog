## if

```sh
if command;then
   ...
elif command;then
   ...
else
   ...
fi
```

- 判断文件是否存在
  
```sh
if ! -f "${path}";then
    ...    
fi
```

| 参数 | 判断是否存在         |
| ---- | -------------------- |
| -e   | 对象                 |
| -d   | 目录                 |
| -f   | 文件                 |
| -L   | 符号链接             |
| -h   | 软链接               |
| -s   | 长度不为0对象        |
| -r   | 可读对象             |
| -w   | 可写对象             |
| -x   | 可执行对象           |
| -O   | 属于当前用户的对象   |
| -G   | 属于当前用户组的对象 |

## case

```sh
case 值 in
模式1)
    ...
    ;;
模式2）
    ...
    ;;
esac
```

## for

- 累积循环

```sh
for i in {start..end}
```

```sh
for i in {1..5};do
    echo ${i}
done
```

- 遍历

```sh
for i in 参数1 参数2 参数3
```

```sh
for i in This is a string;do
    echo ${i}
done
```

## while

```sh
i=1

while((${i}<=5));do
    echo ${i}
    let "i++" 
done
```
