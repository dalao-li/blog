## 变量

```sh
# 定义
a=123

# 使用
echo ${a}
```

- 只读变量

```sh
readonly v
```
 
```sh
#!/bin/bash

# 例，获取VLC的pid，关闭VLC
pid=${ps -aux | grep vlc | grep -v grep | awk '{print $2}'}

if ! "${pid}";then
    sudo kill -9 ${pid}
fi
```

## 字符串

- 单引号

```sh
s='this is a string'
```

单引号里的任何字符都会原样输出，单引号字符串中的变量无效

单引号字串中不能出现单独一个的单引号(对单引号使用转义符后也不行)，但可成对出现，作为字符串拼接使用

- 双引号

```sh
n="runoob"

s="Hello, I know you are "${n}""
```

```sh
# 获取字符串长度
${#str}

# 从字符串index下标开始截取n个字符
${str:index:n}
```

## 大小写转换

- 转小写

```sh
# 变量中首字符换成小写  
${变量名,}

# 所有大写字母，全部替换为小写
${变量名,,}
```

- 转大写

```sh
# 变量中首字符换成大写
${变量名^}

# 所有小写字母全部替换为大写
${变量名^^}
```


## 判断变量是否空

- 直接判断
```sh
if ! ${n};then
...
fi
```

- test命令判断
  
```sh
if test -z "${n}";then
...
fi
```

- 空判断
  
```sh
if "${n}" = "";then
...
fi
```
