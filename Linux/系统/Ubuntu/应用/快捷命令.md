## 快捷指令

```sh
# alias 新指令='原本指令'

sudo bash -c "cat > ~/.bashrc" <<EOF
alias update='sudo apt-get update && sudo apt-get upgrade -y'
alias clean="sudo sh -c 'echo 3 > /proc/sys/vm/drop_caches'"
EOF

# 刷新
source ~/.bashrc
```
