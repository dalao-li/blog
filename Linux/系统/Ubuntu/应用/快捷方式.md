## 创建快捷方式

```sh
name=${1}

app="/usr/local/bin/${name}/${name}.AppImage"

icon="/usr/local/bin/${name}/${name}.png"

sudo bash -c "cat > /usr/share/applications/${name}.desktop" <<EOF
[Desktop Entry]

Type=Application
Name=${name}
Exec=${app}
Icon=${icon}
StartupWMClass=AppRun
EOF
```
