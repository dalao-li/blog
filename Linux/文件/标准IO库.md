## 标准I/O库


### fopen

```c
FILE *fopen(const char *filename , const char *mode);
```

| mode参数 | 含义                           |
| -------- | ------------------------------ |
| r/rb     | 只读方式打开                   |
| w/wb     | 只写方式打开                   |
| a/ab     | 写方式打开，新内容追加在文件尾 |
| r+/rb+   | 读写方式打开                   |
