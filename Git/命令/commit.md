## commit

### 提交

- 将暂存区里的改动给提交到本地的版本库

使用编辑器添加注释 `git commit`

直接添加注释 `git commit -m "提交的注释信息"`

![](https://cdn.hurra.ltd/img/20220112081127.png)


### 修改

- 修改最近一次 commit 

`git commit --amend`

- 修改倒数第三条commit 

`git rebase -i HEAD~3`
