## init

- 新建代码仓库(当前目录)

```sh
git init
```

- 新建代码仓库(指定目录)
  
```sh
git init 项目名
```

## status

- 查看文件状态

```sh
git status
```

![](https://cdn.hurra.ltd/img/20220112080241.png)

## add

- 添加所有文件到暂存区

```sh
git add .
```

- 撤回暂存区文件

```sh
# 默认所有文件，也可指定文件
git reset HEAD
```

![](https://cdn.hurra.ltd/img/20220112080614.png)

## log

提交记录

```sh
git log
```
