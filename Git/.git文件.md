后面的--$prune=now$ 表示对之前的所有提交做修剪，有的时候仅仅$gc$一下.git 文件就会小很多）

```sh
git gc --prune=now
```

```sh
git rev-list --objects --all | grep "$(git verify-pack -v .git/objects/pack/*.idx | sort -k 3 -n | tail -5 | awk '{print$1}')"
```
