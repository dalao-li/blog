# branch

## 查看当前分支

```sh
git branch
```

## 查看所有分支

```sh
git branch -a
```

![](https://cdn.hurra.ltd/img/20220112081438.png)

## 建立分支

```sh
git branch [分支名]
```

## 删除分支

```sh
git branch -d [分支名]
```
