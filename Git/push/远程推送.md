## push

从将本地的分支版本上传到远程并合并

```sh
git push 远程主机名 本地分支名:远程分支名
```

上传本地分支到远程分支

```sh
git checkout -b 分支

# 建立本地到上游(远端)仓的链接，这样代码才能提交上去
git branch --set-upstream-to=origin/分支

git push origin 分支
```
