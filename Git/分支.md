# Git branch


## 查看


### 当前使用分支

```sh
git branch
```


### 所有分支

```sh
git branch -a
```


![](https://cdn.hurra.ltd/img/20220112081438.png)


## 建立

```
git branch 分支名
```

### 切换

#### 切换到分支

```sh
git checkout 分支名
```

#### 创建新分支并切换

```sh
git checkout -b 分支名
```


## 删除


```sh
git branch -d 分支名
```


## 合并


### 合并A分支到当前分支

```sh
git merge A分支
```
