## 浮点数


### 判断相等


```c
int a;
double b;

// 向下取整
printf("%.0lf\n", floor(a * b * 0.1));
```


### 相除余数


```c
double a, b;

scanf("%lf%lf", &a, &b);

int k = a / b;
double r = a - k * b;

printf("%g", r);
```
