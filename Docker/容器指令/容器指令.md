## 重命名

```sh
docker tag 镜像ID 仓库:标签
```

## 删除

```sh
# 删除所有
docker rm $(docker ps -aq)

# 停用删除
docker stop $(docker ps -aq) && docker rm $(docker ps -aq)
```
