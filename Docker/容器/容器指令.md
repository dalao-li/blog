## 容器信息

```sh
# IP地址
docker inspect 容器id/容器名 | grep IPAddress
```

## 打包加载

```sh
# 打包
docker export 容器名/ID > 压缩包.tar

# 加载
docker import 压缩包.tar 容器:tag
```

## 重命名

```sh
docker tag 镜像ID 仓库:标签
```

## 删除

```sh
# 删除所有
docker rm $(docker ps -aq)

# 停用删除
docker stop $(docker ps -aq) && docker rm $(docker ps -aq)
```
