## 目录映射

```sh
# 容器目录只能为绝对路径(必须以下斜线/开头)
docker -v 宿主机目录:容器目录(:权限)

# 例，docker -v $PWD/data:/data:rw ....
```
| 权限 | 含义 |
| ---- | ---- |
| rw   | 读写 |
| ro   | 只读 |
