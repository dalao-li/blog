## 继承

```py
class People:
    def __init__(self, name: str, age: str)-> None:
        self.name = name
        self.age = age

    def speak(self):
        print(self.name, self.age)
```

- 继承类

```py
class Student(People):
    def __init__(self, name: str, age:str, num: str)-> None:
        People.__init__(self, name, age)
        self.num = num
    
    def speak(self):
        print(self.name, self.age, self.num)
```
