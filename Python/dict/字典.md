## 字典

字典是另一种可变容器模型，且可存储任意类型对象

字典的每个键值以`:`分割，每个键值对之间用`,`分割，字典包括在花括号 {} 中


```py
d = {'Name': 'Zara', 'Age': 7, 'Class': 'First'}
```
## 访问

- 键值

```py
# 7
print(d['Age'])
```

- 遍历

```py
for k , v in d.items():
    print(k, v)
```

## 修改
  
```py
d[k] = v
```

## 删除

```py
# 删除键是'Name'的条目
del d['Name']

# 清空字典所有条目
d.clear()

# 删除字典
del d         
```
