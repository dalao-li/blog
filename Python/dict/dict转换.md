## dict json

在Python中通过import json导入

| 函数    | 作用                            |
| ------- | ------------------------------- |
| loads() | 将json转化成dict                |
| dumps() | 将dict转化成json                |
| load()  | 读取json，转成dict              |
| dump()  | 将dict转化成json后写入jsonw文件 |
