## 二维数组

```py
# 建立 x*y 大小的数组
[[0 for i in range(x)] for j in range(y)]
```

```py
a = [[0 for i in range(3)] for j in range(3)]

a[0][0] = 1

a[1][1] = 3

print(a[1][1])

print(a)
```
