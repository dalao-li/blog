## 二维数组

- 建立 $x*y$ 大小的数组，初始值为 $z$

```py
[[z for i in range(x)] for j in range(y)]
```

建立3*3二维数组，初始值0

```py
a = [[0 for i in range(3)] for j in range(3)]

a[0][0] = 1

a[1][1] = 3

# 3
print(a[1][1])

# [[1, 0, 0], [0, 3, 0], [0, 0, 0]]
print(a)
```
