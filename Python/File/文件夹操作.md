## 显示文件

```py
def display_all_files(folder_path):
    for i in os.listdir(folder_path):
        path = os.path.join(folder_path， i)
        # 如果是文件则显示
        if os.path.isfile(path):
            print(path)
```

## 递归显示

```py
def display_all_folders(folder_path):
    for i in os.listdir(folder_path):
        path = os.path.join(folder_path， i)
        # 如果该对象是文件夹
        if os.path.isdir(path):
            print("--folder : %s"%i)
            display_all_files(path)
```
