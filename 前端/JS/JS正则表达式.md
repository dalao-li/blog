# 校验数字的表达式

- 数字

`^[0-9]*$ `
  
- n 位的数字

`^\d{n}$`
  
- 至少 n 位的数字

`^\d{n，}$`
  
- m-n 位的数字

`^\d{m，n}$`
  
- 零和非零开头的数字

`^(0|[1-9][0-9]\*)$`
  
- 带 1-2 位小数的正数或负数

`^(\-)?\d+(\.\d{1，2})$`
  
- 正数、负数、和小数

`^(\-|\+)?\d+(\.\d+)?$`
  
- 有 1~3 位小数的正实数

`^[0-9]+(\.[0-9]{1，3})?$`
  
- 非零的正整数

`^[1-9]\d*$ 或 ^([1-9][0-9]*){1，3}$` 或 `^\+?[1-9][0-9]\*$`
  
- 非零的负整数

`^\-[1-9][]0-9"_$` 或 `^-[1-9]\d_$`
  
- 非负整数

`^\d+$` 或 `^[1-9]\d\*|0$`
  
- 非正整数

`^-[1-9]\d\*|0$` 或 `^((-\d+)|(0+))$`
  
- 浮点数

`^(-?\d+)(\.\d+)?$` 或 `^-?([1-9]\d*\.\d*|0\.\d*[1-9]\d*|0?\.0+|0)$`
