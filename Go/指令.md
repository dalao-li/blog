# Go指令


```go
// main.go
package main

import (
	"fmt"
)

func main() {
	fmt.Println("Hello World!")
}
```


## 运行


```go
go run main.go
```


## 编译

```go
go build main.go
```


## 命令行参数

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	for i := 1; i < len(os.Args); i++ {
		fmt.Println(os.Args[i])
	}
}
```
